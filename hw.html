Write a function named computeSumOfSquaresOfEvensOnly that takes as input, an array of integral numbers and calculates and returns the sum of the squares of only the even numbers in the input array. function computeSumOfSquaresOfEvensOnly(numbers) { return numbers.filter(num => num % 2 === 0).reduce((acc, evenNum) => acc + evenNum ** 2, 0); } // Example usage: console.log(computeSumOfSquaresOfEvensOnly([1, 2, 3, 4, 5])); // Output: 20 // 11.
